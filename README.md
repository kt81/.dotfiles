.dotfiles
=======

Overview
--------

My dotfile collection for:

- Windows (Powershell 7+)
- Ubuntu (Windows Subsystem for Linux)
- macOS

Including:

- Setup script
- Nice shell prompt
- IDE-like neovim
- powerline-like tmux theme
- Build tools
- asdf
- cargo
- dotnet
- powershell

Usage
------

```shell
cd ~
git clone https://github.com/kt81/.dotfiles.git
.dotfiles/setup.sh
```
